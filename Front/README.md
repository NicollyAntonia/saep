# SAEP - Frontend (React + Vite)

Projeto front-end de exemplo para a prova SAEP (sistema de almoxarifado).

## Como usar

1. Instale dependências:
   ```
   npm install
   ```

2. Execute em desenvolvimento:
   ```
   npm run dev
   ```

3. Ajuste `src/api/api.js` para apontar para seu back-end (baseURL).

## Estrutura
- src/pages: Login, Home, Produtos, Estoque
- src/api/api.js: configuração do Axios
